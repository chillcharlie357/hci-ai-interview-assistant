# MVP 目标

## 产品目标

实现一个可演示的面试智能辅助 MVP，覆盖以下闭环：

1. 输入候选人简历摘要、岗位 JD、面试目标。
2. 系统解析输入并生成结构化面试问题。
3. 数字人面试官按问题逐个提问。
4. 系统记录候选人回答、表达指标和关键事件。
5. 面试结束后生成完整智能纪要。

## MVP 范围

第一版聚焦浏览器内可运行的轻量 Demo，不接入真实 WebRTC 会议和实时数字人渲染。会议、数字人和实时分析均用可替换接口封装，后续可接入 LiveKit、LiveKit Agents、Pipecat 或 LiveTalking。

## 核心成功标准

- 可以从简历/JD生成不少于 6 个面试问题。
- 问题覆盖技术能力、项目经验、实现能力、行为表现和追问。
- 数字人面试官可以用文本和浏览器 TTS 逐题提问。
- 可以记录每道题的回答内容、用时、字数和填充词数量。
- 可以在面试后生成 Markdown 风格纪要。
- 核心逻辑有自动化测试覆盖。

## 非目标

- 不在 MVP 中实现真实视频会议。
- 不在 MVP 中实现真实数字人唇形同步。
- 不在 MVP 中实现摄像头非语言识别。
- 不在 MVP 中实现真实视频抽帧、OCR 和 WhisperX。
- 不输出录用或不录用结论。
